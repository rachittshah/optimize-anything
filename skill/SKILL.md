# optimize-anything

Use the optimize-anything MCP tools to optimize text artifacts.

## Quick Start

1. Formulate your optimization problem as (candidate, evaluator, objective)
2. Call `optimize_anything` tool with seed candidate and evaluator config
3. Monitor with `check_optimization`
4. Retrieve results with `get_best_candidate`
